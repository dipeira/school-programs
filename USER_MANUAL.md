# Εγχειρίδιο Χρήστη (Σχολικές Μονάδες)
## Προγράμματα Σχολικών Δραστηριοτήτων

Το σύστημα αυτό επιτρέπει στις σχολικές μονάδες να καταχωρούν, να επεξεργάζονται και να εκτυπώνουν τα Προγράμματα Σχολικών Δραστηριοτήτων τους.

### 1. Είσοδος στο Σύστημα
Η είσοδος γίνεται αποκλειστικά με τους επίσημους κωδικούς του **Πανελληνίου Σχολικού Δικτύου (ΠΣΔ)** της σχολικής μονάδας (π.χ. `1234567@sch.gr`).
*Σημείωση: Δεν χρησιμοποιούνται οι κωδικοί του MySchool.*

### 2. Διαχείριση Προγραμμάτων
Μετά την είσοδο, μπορείτε να δείτε τη λίστα με τα προγράμματα του σχολείου σας.
- **Προσθήκη:** Πατήστε το κουμπί "Προσθήκη" για να δημιουργήσετε ένα νέο πρόγραμμα.
- **Επεξεργασία:** Πατήστε "Επεξεργασία" σε μια υπάρχουσα εγγραφή για να αλλάξετε τα στοιχεία (Τίτλος, Εκπαιδευτικοί, Μαθητές κ.λπ.).
- **Προβολή:** Πατήστε "Προβολή" για να δείτε τα συγκεντρωτικά στοιχεία της εγγραφής.

### 3. Εκτύπωση και Βεβαιώσεις
- **Εκτύπωση Προγράμματος (A4):** Μέσα στην καρτέλα επεξεργασίας κάθε προγράμματος, υπάρχει το κουμπί **"Εκτύπωση (A4)"**. Αυτό παράγει ένα έγγραφο έτοιμο για εκτύπωση με όλα τα στοιχεία του προγράμματος.
- **Βεβαίωση Υλοποίησης:** Όταν η Διεύθυνση ελέγξει και εγκρίνει το πρόγραμμα (Κατάσταση: **Ναι**), θα ενεργοποιηθεί το κουμπί **"Βεβαίωση"**. Πατώντας το, κατεβαίνει αυτόματα το επίσημο έγγραφο της βεβαίωσης σε μορφή Word (.docx).

### 4. Ιστορικό Αρχείο
Μπορείτε να δείτε προγράμματα παλαιότερων ετών επιλέγοντας το αντίστοιχο έτος από το μενού **"Επιλογή Έτους"**. Στο αρχείο, η προβολή είναι μόνο για ανάγνωση.
